div class="container"

Principalmente para poder crear una rejilla es poner un div y una class container como se puede observar se ha creado una estructura para poner en práctica lo que es el sistema de rejilla con un div class”container”por lo cual nuestra estructura se centra



div class="row"

Luego se usa lo que es un div con la class row que significa fila en español di class”row” luego se organiza de manera horizontal lo que sería nuestro contenido y gracias a flexbox, practica mente se activa nuestro display flex es así como se puede observar lo que son las fila de una manera adaptable para cualquier dispositivo.



div class=".col-"

En lo que son las columnas se le agregaron las class con el prefijo col-sm, col-md, col-lg y col-xl para ser consecuente con la filosofía de boostrap y poder ser adaptable a cualquier dispositivo.
